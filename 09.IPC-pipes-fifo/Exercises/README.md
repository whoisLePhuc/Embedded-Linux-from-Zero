# BÀI TẬP VỀ IPC PIPES & FIFOS

Bộ bài tập gồm 6 bài từ cơ bản đến nâng cao, giúp bạn làm chủ Pipe và Named Pipe.

## 📚 Danh sách bài tập

1.  **exercise-01-pipe**: Anonymous Pipe cơ bản giữa Parent và Child ⭐⭐
2.  **exercise-02-fifo**: Named Pipe giữa 2 process độc lập ⭐⭐
3.  **exercise-03-popen-pclose**: Sử dụng popen/pclose để chạy shell command ⭐⭐
4.  **exercise-04-redirection**: Mô phỏng Shell Pipe (`ls | wc -l`) với pipe+fork+dup2 ⭐⭐⭐
5.  **exercise-05-client-server**: Client-Server với FIFO ⭐⭐⭐
6.  **exercise-06-pipe-nonblock**: Non-blocking I/O trên pipe ⭐⭐⭐
