# BÀI TẬP VỀ MAKEFILE

Bộ bài tập này được thiết kế để giúp bạn làm quen với Makefile từ cơ bản đến nâng cao.

## 📚 Cách sử dụng

Mỗi bài tập được đặt trong một thư mục riêng:
- `exercise-01-basic/` - Makefile cơ bản (1 file C) ⭐
- `exercise-02-multi-files/` - Biên dịch nhiều file C ⭐⭐
- `exercise-03-library/` - Thư viện tĩnh (.a) ⭐⭐
- `exercise-04-shared-library/` - Thư viện động (.so) ⭐⭐⭐
- `exercise-05-advanced/` - Makefile nâng cao (VPATH, conditional) ⭐⭐⭐
- `exercise-06-makefile-functions/` - Functions và wildcards ⭐⭐⭐
- `exercise-07-conditional-build/` - Conditional build và multi-config ⭐⭐⭐

## 🎯 Hướng dẫn

1. Đọc file `README.md` trong mỗi thư mục bài tập
2. Làm theo yêu cầu để hoàn thành bài tập
3. Kiểm tra kết quả bằng cách chạy `make` và `make test` (nếu có)
4. So sánh với file `solution/Makefile` nếu cần gợi ý

## 📝 Mức độ

- ⭐ Dễ: Exercise 01
- ⭐⭐ Trung bình: Exercise 02, 03
- ⭐⭐⭐ Khó: Exercise 04, 05, 06, 07

## 💡 Tips

- Đọc kỹ comment trong code mẫu
- Sử dụng `make -n` để xem lệnh sẽ được thực thi
- Sử dụng `make -B` để force rebuild tất cả
- Nhớ rằng Makefile yêu cầu TAB, không phải SPACE!

Chúc bạn học vui!
